# Category: Input / Output (IO)

Modules in this category provide contact with the world outside a modular rack, eg. by importing audio or CV signals from external sources and making it safe for use within the 5V circuitry of the AE Modular. Another use for IO modules is to provide line level or amplified audio output to external mixing desks, recorders or other gear.
